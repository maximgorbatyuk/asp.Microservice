# asp.Microservice

Here is an example of microservice application.

## Libs

- MediatR
- FluentValidation
- AutoMapper
- PostgreSQL (Npgsql)

## Approaches

- CQRS
- DDD
- Clean Architecture

## How to start the development

1. Install .NET 7 SDK
2. Install Docker
3. Run `docker-compose up -d` to start the app and services
4. Open `http://localhost:7278/swagger/index.html` in your browser
5. Have fun.
